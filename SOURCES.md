# Sources et licences

Ce fichier sert a documenter les ressources externes utilisees dans le jeu.

Pour chaque image, police, icone, son ou autre element pris sur internet, ajouter une ligne avec:

- Titre de la ressource
- Auteur ou organisation
- URL source
- Licence
- Attribution demandee
- Date de consultation

## Ressources actuellement utilisees

Aucune ressource externe n'est integree pour le moment. Les formes, couleurs et elements visuels du jeu sont crees en HTML/CSS.

## Exemple d'attribution

`Nom de la ressource` par `Auteur`, source: `URL`, licence: `CC BY 4.0`.

## Regle simple pour le cours

Avant d'utiliser une ressource internet dans un projet HTML/CSS:

1. Verifier la licence.
2. Noter la source.
3. Ajouter l'attribution si la licence le demande.
4. Eviter les contenus sans autorisation claire.
